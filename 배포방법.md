# 몰랑이 💗 (내부 Arcogit + 외부 몰랑이)

## 구조
- **내부**: Arcogit 전체가 돎 (사고계보·자기검증·양심·기억·유형생성 20종)
- **외부**: 몰랑이 껍데기 (말투·표정·카톡 UI)
- **pkl**: 다운로드/업로드로 정체성 보관·이어가기
  (Streamlit 무료 플랜 잠자기 문제를 다운/업로드로 회피)

## 파일
```
app.py              카톡 UI + 채팅
molang_skin.py      몰랑이 껍데기 (표정/외형/말투)
molang_persist.py   pkl 번들 저장·복원 (Arcogit+표정 하나로)
── 아래는 Arcogit 엔진 (그대로) ──
unified_identity.py / thought_structure.py / tree_registry.py
identity_core.py / conscience.py / llm_bridge.py / logic_db.json
```

## 배포
1. **GitHub 업로드** — secrets.toml 빼고 전부 (.gitignore가 막아줌)
2. **Streamlit Cloud** — share.streamlit.io → New app → app.py
3. **Secrets에 API 키**:
   ```
   OPENAI_API_KEY = "sk-본인키"
   ```
   (본인 키 넣으면 여친은 키 없이 사용)

## 몰랑이 세팅 (배포 후 1회)
사이드바에서:
1. 몰랑이 사진 업로드 → **① 외형 학습** (GPT-4V가 특징 고정)
2. **② 표정 5종 생성** (기쁨/슬픔/뾰로통/놀람/사랑, 1~2분)
3. **⬇️ 몰랑이 저장** 눌러 pkl 다운로드 → 보관!

## 여친이 쓰는 법
- 링크 열고 대화 → 몰랑이 프로필이 감정 따라 바뀜
- 대화할수록 Arcogit이 진화 (기억·정체성 축적)
- 끝낼 때 **⬇️ 몰랑이 저장**으로 pkl 받아두면
- 다음에 **💾 불러오기**로 그 몰랑이 이어서

## 핵심
- 표정은 **일관성 유지**하며 생성 (image_gascore 방식: 외형 고정 후 표정만)
- 한 번 만든 표정은 pkl에 캐시 → 다음엔 생성 없이 즉시
- 사진 올리면 몰랑이가 보고 반응 (png/jpg/webp/gif/bmp)

## 주의
- API 키 절대 GitHub에 X
- 몰랑이는 실제 캐릭터 → 개인 선물용만
- pkl은 다운/업로드로 관리 (무료 플랜은 세션 종료 시 초기화되니 꼭 저장)
